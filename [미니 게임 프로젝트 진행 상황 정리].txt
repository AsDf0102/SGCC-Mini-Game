[미니 게임 프로젝트 진행 상황 정리]

02.10(1일차) 
    - 개발 시작
    - 구현 단계
        1. 플레이어 이동 : 키보드로 Player 움직이기
        2. 화면 이탈 방지 : 화면 밖으로 움직이지 않도록 이동 제한 걸기
        3. 장애물, 아이템 자동 생성 : 화면 밖 랜덤한 위치에서 장애물이나 아이템이 쏟아지도록 구현
        4. 장애물 충돌
            1). 라이프 시스템 : 부딪히면 라이프 감소
            2). 충돌 효과 : 충돌 시 1초 동안 무적, 색상 변경 -> 1초 후 원래 상태로 복구
        5. 아이템 충돌
            1). 점수 증가
        6. UI 구축
            1). ScoreText : 현재 점수 표시  
            2). TimerText : 남은 시간 표시
        7. 게임 종료
            1). 정해진 2분의 시간이 지나면 게임 종료

            3). 에셋 교체 : 실제 이미지 추가
